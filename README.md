**Installation**
1) Install Git
>  - *sudo apt install git*
  
2) Download from this repository
>  - *git clone https://github.com/CandyScriptz/WhosHome*
  
3) Give permission to file
>  - *chmod +x WhosHome.py*
 
4) Run program
>  - *./WhosHome.py*
